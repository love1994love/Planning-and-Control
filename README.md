

## 如何使用

1. 克隆仓库

   ```terminal
   git clone https://github.com/love1994love/Planning-and-Control.git
   ```


2. 安装依赖库

- conda :

  ```terminal
  conda env create -f requirements/environment.yml
  ```

- pip :

  ```terminal
  pip install -r requirements/requirements.txt
  ```
3. enjoy it! :smiley:
